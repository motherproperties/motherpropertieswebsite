## Image Guidelines

- Use high-quality, royalty-free images from sources like Unsplash or Pexels
- Optimize images for web (compress without losing quality)
- Recommended formats: JPG for photos, PNG for logos with transparency
- Maintain consistent aspect ratios for similar image types
